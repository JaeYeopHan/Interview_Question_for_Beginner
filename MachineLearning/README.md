## Cost Function
#### [ 비용 함수 (Cost Function) ]
**Cost Function**이란 **데이터 셋**과 어떤 **가설 함수**와의 오차를 계산하는 함수이다.<br/>
Cost Function의 결과가 작을수록 데이터셋에 더 **적합한 Hypothesis**(가설 함수)라는 의미다.<br/>
**Cost Function**의 궁극적인 목표는 **Global Minimum**을 찾는 것이다.
